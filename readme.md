## TRADEUP

### Teste para Mobile Developer


**Objetivo**: Desenvolver um CRUD com autenticação e duas páginas que exibem os dados providos de uma API Rest.


Essa aplicação foi desenvolvida com [React-Native](https://reactnative.dev) e bibliotecas como: [Axios](https://github.com/axios/axios), [Styled-Components](https://styled-components.com), [React-Navigation](https://reactnavigation.org) e outras que podem ser consultadas no *package.json*.

Para tipagem, foi utilizado [Typescript](https://www.typescriptlang.org).

### Instalação
Clone o repositório:
```sh
git clone https://github.com/devmauromoura/TesteTradeup.git
```
Abra o diretório e instale as dependências. 
```sh
npm install 
//ou
yarn install
```

### Execução
Para executar a aplicação em DEBUG, abra o emulador ou conecte o dispositivo e execute:
```sh
//Android
npx react-native run-android

//iOS
npx react-native run-ios
```
Para executar/instalar em Release, execute:
```sh
//Android 
npx react-native run-android --variant=release

//iOS
react-native run-ios --configuration Release
```
